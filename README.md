# 📝 TO-DO List App

A simple command-line To-Do List application built with Python.

## Features
- Add new tasks
- View all tasks
- Remove tasks
- Mark tasks as completed
- Save tasks using file handling
- Load tasks automatically when the program starts

## Technologies Used
- Python
- File Handling
- Lists
- Dictionaries
- Loops
- Functions

## How to Run

```bash
python main.py
```
